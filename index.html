Overview

The Unity Save System is a comprehensive, modern save/load solution for Unity games. Built with JSON serialization at its core, it provides a clean API for saving game data with support for encryption, compression, and multiple save slots.

üöÄ Easy to Use

Simple API with just Save() and Load<T>() methods. Get started in minutes with minimal setup required.

üîí Secure

Built-in AES encryption support to protect save data from tampering and cheating.

üì¶ Compressed

GZIP compression reduces file sizes significantly, improving load times and storage efficiency.

üéÆ Unity Ready

Full support for all Unity types including Vector3, Color, Quaternion, AnimationCurve, and more.

üìÅ Multi-Slot

Built-in save slot management allows multiple save files with easy switching between them.

‚ö° Async

Modern async/await pattern prevents frame drops during save/load operations.

Quick Start

1. Add Setup Wizard

Add the SetupWizard component to any GameObject in your first scene:

C# 
using SaveLoadSystem;
using UnityEngine;

public class GameInitializer : MonoBehaviour
{
    private void Awake()
    {
        // Create or use existing save slot
        if (!DataManager.DoesSlotExist("MainSave"))
        {
            DataManager.CreateSaveSlot("MainSave");
        }
        DataManager.SetActiveSlot("MainSave");
    }
}

2. Save Data

C# 
// Save simple values
DataManager.Save("PlayerLevel", 42);
DataManager.Save("PlayerName", "John Doe");
DataManager.Save("PlayerPosition", transform.position);

// Save complex objects
var playerData = new PlayerData
{
    Health = 100,
    Experience = 1500,
    Inventory = new List<string> { "Sword", "Potion", "Key" }
};
DataManager.Save("PlayerData", playerData);

3. Load Data

C# 
// Load simple values
int level = DataManager.Load<int>("PlayerLevel");
string name = DataManager.Load<string>("PlayerName");
Vector3 position = DataManager.Load<Vector3>("PlayerPosition");

// Load complex objects
PlayerData playerData = DataManager.Load<PlayerData>("PlayerData");
if (playerData != null)
{
    // Use loaded data
    health = playerData.Health;
    experience = playerData.Experience;
}

Features

Supported Data Types

  • Primitives: int, float, string, bool, etc.
  • Collections: List<T>, Dictionary<K,V>, arrays
  • Unity Types: Vector2/3/4, Quaternion, Color, Rect, Bounds, etc.
  • Custom Classes: Any serializable class or struct
  • Advanced Unity: AnimationCurve, Gradient, Matrix4x4

Save Slot Features

  • Multiple save slots with unique names
  • Runtime save slot creation/deletion
  • Save slot renaming
  • Automatic slot switching

Performance Features

  • Async operations to prevent frame drops
  • GZIP compression for smaller file sizes
  • Atomic write operations to prevent corruption
  • Efficient JSON serialization

DataManager API

The DataManager is the main entry point for the save system. It provides a simple, static API for all save/load operations.

Method Description Example
Save(string key, object value) Saves data to the active save slot DataManager.Save("Score", 1000);
Load<T>(string key) Loads data from the active save slot int score = DataManager.Load<int>("Score");
CreateSaveSlot(string name) Creates a new save slot DataManager.CreateSaveSlot("Player1");
DeleteSaveSlot(string name) Deletes an existing save slot DataManager.DeleteSaveSlot("Player1");
SetActiveSlot(string name) Changes the active save slot DataManager.SetActiveSlot("Player2");
DoesSlotExist(string name) Checks if a save slot exists bool exists = DataManager.DoesSlotExist("Player1");
GetSaveSlots() Returns all available save slots List<SaveSlot> slots = DataManager.GetSaveSlots();

SaveSlot Class

The SaveSlot class represents an individual save file. Each slot can have its own encryption and compression settings.

Properties

Property Type Description
Name string The display name of the save slot
SavePath string Full file path to the save file
UseCompression bool Whether to use GZIP compression
UseEncryption bool Whether to use AES encryption
EncryptionKey byte[] The encryption key (auto-generated if null)

Methods

Method Description
Save(string key, object value) Saves data to this specific slot
Load<T>(string key) Loads data from this specific slot
Rename(string newName) Renames the save slot and updates file path
GetFileSize() Returns the size of the save file in bytes
GetLastModified() Returns when the save file was last modified

Encryption

The save system includes AES encryption to protect save data from tampering. Encryption is applied before compression.

Enabling Encryption

C# 
// Enable encryption when creating a save slot
SaveSlot encryptedSlot = new SaveSlot("SecureData", useCompression: false, useEncryption: true);

// Or modify an existing slot
var activeSlot = DataManager.GetSaveSlots().First();
activeSlot.UseEncryption = true;
activeSlot.EncryptionKey = EncryptionUtility.GenerateKey(); // Auto-generated if null

Encryption Utilities

Method Description
EncryptionUtility.GenerateKey() Generates a new random AES key
EncryptionUtility.Encrypt(byte[] data, byte[] key) Encrypts data with the provided key
EncryptionUtility.Decrypt(byte[] data, byte[] key) Decrypts data with the provided key
Important: Store encryption keys securely! If the key is lost, save data cannot be recovered.

Compression

GZIP compression can significantly reduce save file sizes, especially for text-heavy data like JSON.

Enabling Compression

Compression Performance

üìä Typical Savings

JSON text compresses by 60-80% on average, with larger files seeing better compression ratios.

‚ö° Load Times

Smaller files mean faster disk I/O, often outweighing decompression overhead.

Compression Utilities

Method Description
GZIP.Compress(string text) Compresses a string to byte array
GZIP.Decompress(byte[] compressed) Decompresses byte array back to string

Error Recovery

C# 
// Load with fallback values
int playerLevel = DataManager.Load<int>("PlayerLevel");
if (playerLevel == 0) // Default value indicates load failure
{
    playerLevel = 1; // Fallback to level 1
    DataManager.Save("PlayerLevel", playerLevel); // Save corrected value
}

// Check if data exists before loading complex objects
if (DataManager.DoesSlotExist("MainSave"))
{
    PlayerData data = DataManager.Load<PlayerData>("PlayerData");
    if (data != null)
    {
        // Successfully loaded
        ApplyPlayerData(data);
    }
    else
    {
        // Failed to load, create new data
        CreateNewPlayerData();
    }
}
Atomic Operations: All save operations use atomic writes (temp file ‚Üí rename) to prevent corruption if the game crashes during saving.

Basic Examples

Player Progression System

C# 
using SaveLoadSystem;
using UnityEngine;

[System.Serializable]
public class PlayerStats
{
    public int Level = 1;
    public int Experience = 0;
    public int Health = 100;
    public int Mana = 50;
    public Vector3 LastPosition = Vector3.zero;
}

public class PlayerController : MonoBehaviour
{
    [SerializeField] private PlayerStats stats;
    
    private void Start()
    {
        // Initialize save system
        if (!DataManager.DoesSlotExist("PlayerSave"))
        {
            DataManager.CreateSaveSlot("PlayerSave");
        }
        DataManager.SetActiveSlot("PlayerSave");
        
        // Load player stats
        LoadPlayerData();
    }
    
    private void LoadPlayerData()
    {
        stats = DataManager.Load<PlayerStats>("PlayerStats");
        if (stats == null)
        {
            stats = new PlayerStats(); // Create default stats
            SavePlayerData(); // Save defaults
        }
        
        // Apply loaded data
        transform.position = stats.LastPosition;
        UpdateUI();
    }
    
    private void SavePlayerData()
    {
        stats.LastPosition = transform.position;
        DataManager.Save("PlayerStats", stats);
    }
    
    public void GainExperience(int amount)
    {
        stats.Experience += amount;
        CheckLevelUp();
        SavePlayerData(); // Auto-save on progression
    }
    
    private void CheckLevelUp()
    {
        int requiredExp = stats.Level * 100;
        if (stats.Experience >= requiredExp)
        {
            stats.Level++;
            stats.Health += 10; // Level up bonus
            stats.Mana += 5;
            Debug.Log($"Level up! Now level {stats.Level}");
        }
    }
    
    private void OnApplicationPause(bool pauseStatus)
    {
        if (pauseStatus) SavePlayerData(); // Save when app is paused
    }
}

Settings System

C# 
using SaveLoadSystem;
using UnityEngine;
using UnityEngine.Audio;

[System.Serializable]
public class GameSettings
{
    public float MasterVolume = 1.0f;
    public float SFXVolume = 1.0f;
    public float MusicVolume = 1.0f;
    public int QualityLevel = 2;
    public bool Fullscreen = true;
    public Resolution Resolution = new Resolution { width = 1920, height = 1080 };
    public KeyCode JumpKey = KeyCode.Space;
    public KeyCode InteractKey = KeyCode.E;
}

public class SettingsManager : MonoBehaviour
{
    [SerializeField] private AudioMixer audioMixer;
    [SerializeField] private GameSettings settings;
    
    public static SettingsManager Instance { get; private set; }
    
    private void Awake()
    {
        if (Instance == null)
        {
            Instance = this;
            DontDestroyOnLoad(gameObject);
            LoadSettings();
        }
        else
        {
            Destroy(gameObject);
        }
    }
    
    private void LoadSettings()
    {
        // Use a dedicated settings slot
        if (!DataManager.DoesSlotExist("Settings"))
        {
            DataManager.CreateSaveSlot("Settings");
        }
        
        var currentSlot = DataManager.GetSaveSlots().FirstOrDefault(s => s.Name == "Settings");
        if (currentSlot != null)
        {
            DataManager.SetActiveSlot("Settings");
            settings = DataManager.Load<GameSettings>("GameSettings");
        }
        
        if (settings == null)
        {
            settings = new GameSettings(); // Use defaults
        }
        
        ApplySettings();
    }
    
    public void SaveSettings()
    {
        DataManager.SetActiveSlot("Settings");
        DataManager.Save("GameSettings", settings);
    }
    
    private void ApplySettings()
    {
        // Audio
        audioMixer.SetFloat("MasterVolume", Mathf.Log10(settings.MasterVolume) * 20);
        audioMixer.SetFloat("SFXVolume", Mathf.Log10(settings.SFXVolume) * 20);
        audioMixer.SetFloat("MusicVolume", Mathf.Log10(settings.MusicVolume) * 20);
        
        // Graphics
        QualitySettings.SetQualityLevel(settings.QualityLevel);
        Screen.SetResolution(settings.Resolution.width, settings.Resolution.height, settings.Fullscreen);
    }
    
    // Public methods for UI to call
    public void SetMasterVolume(float volume)
    {
        settings.MasterVolume = volume;
        audioMixer.SetFloat("MasterVolume", Mathf.Log10(volume) * 20);
        SaveSettings();
    }
    
    public void SetQuality(int level)
    {
        settings.QualityLevel = level;
        QualitySettings.SetQualityLevel(level);
        SaveSettings();
    }
}

Inventory System with Custom Types

C# 
using System.Collections.Generic;
using SaveLoadSystem;
using UnityEngine;

[System.Serializable]
public class Item
{
    public string ID;
    public string Name;
    public int Quantity;
    public Color Rarity;
    public Vector2 IconPosition;
    public Dictionary<string, object> Properties = new Dictionary<string, object>();
}

[System.Serializable]
public class Inventory
{
    public List<Item> Items = new List<Item>();
    public int MaxSlots = 30;
    public Vector2Int GridSize = new Vector2Int(6, 5);
    
    public bool AddItem(Item item)
    {
        if (Items.Count >= MaxSlots) return false;
        
        // Try to stack with existing item
        var existingItem = Items.Find(i => i.ID == item.ID);
        if (existingItem != null)
        {
            existingItem.Quantity += item.Quantity;
        }
        else
        {
            Items.Add(item);
        }
        return true;
    }
    
    public bool RemoveItem(string itemID, int quantity = 1)
    {
        var item = Items.Find(i => i.ID == itemID);
        if (item == null || item.Quantity < quantity) return false;
        
        item.Quantity -= quantity;
        if (item.Quantity <= 0)
        {
            Items.Remove(item);
        }
        return true;
    }
}

public class InventoryManager : MonoBehaviour
{
    [SerializeField] private Inventory playerInventory;
    
    private void Start()
    {
        LoadInventory();
    }
    
    private void LoadInventory()
    {
        playerInventory = DataManager.Load<Inventory>("PlayerInventory");
        if (playerInventory == null)
        {
            playerInventory = new Inventory();
            // Add some starting items
            AddStartingItems();
        }
        
        RefreshInventoryUI();
    }
    
    private void AddStartingItems()
    {
        var sword = new Item
        {
            ID = "iron_sword",
            Name = "Iron Sword",
            Quantity = 1,
            Rarity = Color.gray,
            IconPosition = new Vector2(0, 0)
        };
        sword.Properties.Add("Damage", 25);
        sword.Properties.Add("Durability", 100);
        
        var potion = new Item
        {
            ID = "health_potion",
            Name = "Health Potion",
            Quantity = 5,
            Rarity = Color.green,
            IconPosition = new Vector2(1, 0)
        };
        potion.Properties.Add("HealAmount", 50);
        
        playerInventory.AddItem(sword);
        playerInventory.AddItem(potion);
        SaveInventory();
    }
    
    public void SaveInventory()
    {
        DataManager.Save("PlayerInventory", playerInventory);
    }
    
    public void AddItemToInventory(string itemID, int quantity = 1)
    {
        // Create item from database/scriptable object
        var item = CreateItemFromID(itemID);
        item.Quantity = quantity;
        
        if (playerInventory.AddItem(item))
        {
            SaveInventory();
            RefreshInventoryUI();
        }
    }

    {
        // In a real game, this would query an item database
        return new Item { ID = itemID, Name = itemID.Replace("_", " "), Quantity = 1 };
    }

    {

    }

World State Persistence

C# 
using System.Collections.Generic;
using SaveLoadSystem;
using UnityEngine;

[System.Serializable]
public class WorldObject
{
    public string ID;
    public Vector3 Position;
    public Quaternion Rotation;
    public bool IsActive;
    public Dictionary<string, object> State = new Dictionary<string, object>();
}

[System.Serializable]
public class WorldData
{
    public List<WorldObject> Objects = new List<WorldObject>();
    public Dictionary<string, bool> CompletedQuests = new Dictionary<string, bool>();
    public Dictionary<string, int> CollectedItems = new Dictionary<string, int>();
    public float TimeOfDay = 12.0f;
    public int DayNumber = 1;
}

public class WorldManager : MonoBehaviour
{
    [SerializeField] private WorldData worldData;
    [SerializeField] private List<GameObject> persistentObjects;
    
    private void Start()
    {
        LoadWorldState();
        ApplyWorldState();
    }
    
    private void LoadWorldState()
    {
        worldData = DataManager.Load<WorldData>("WorldState");
        if (worldData == null)
        {
            worldData = new WorldData();
            CaptureInitialWorldState();
        }
    }
    
    private void CaptureInitialWorldState()
    {
        // Capture all objects marked as persistent
        foreach (var obj in persistentObjects)
        {
            var persistentObj = obj.GetComponent<PersistentObject>();
            if (persistentObj != null)
            {
                var worldObj = new WorldObject
                {
                    ID = persistentObj.UniqueID,
                    Position = obj.transform.position,
                    Rotation = obj.transform.rotation,
                    IsActive = obj.activeInHierarchy
                };
                
                // Capture custom state
                persistentObj.SaveState(worldObj.State);
                worldData.Objects.Add(worldObj);
            }
        }
        
        SaveWorldState();
    }
    
    private void ApplyWorldState()
    {
        foreach (var worldObj in worldData.Objects)
        {
            var gameObj = FindObjectByID(worldObj.ID);
            if (gameObj != null)
            {
                gameObj.transform.position = worldObj.Position;
                gameObj.transform.rotation = worldObj.Rotation;
                gameObj.SetActive(worldObj.IsActive);
                
                var persistentObj = gameObj.GetComponent<PersistentObject>();
                if (persistentObj != null)
                {
                    persistentObj.LoadState(worldObj.State);
                }
            }
        }
    }
    
    public void SaveWorldState()
    {
        // Update all persistent objects
        foreach (var worldObj in worldData.Objects)
        {
            var gameObj = FindObjectByID(worldObj.ID);
            if (gameObj != null)
            {
                worldObj.Position = gameObj.transform.position;
                worldObj.Rotation = gameObj.transform.rotation;
                worldObj.IsActive = gameObj.activeInHierarchy;
                
                var persistentObj = gameObj.GetComponent<PersistentObject>();
                if (persistentObj != null)
                {
                    worldObj.State.Clear();
                    persistentObj.SaveState(worldObj.State);
                }
            }
        }
        
        DataManager.Save("WorldState", worldData);
    }
    
    private GameObject FindObjectByID(string id)
    {
        return persistentObjects.Find(obj => 
        {
            var persistent = obj.GetComponent<PersistentObject>();
            return persistent != null && persistent.UniqueID == id;
        });
    }
    
    public void CompleteQuest(string questID)
    {
        worldData.CompletedQuests[questID] = true;
        SaveWorldState();
    }
    
    public bool IsQuestCompleted(string questID)
    {
        return worldData.CompletedQuests.ContainsKey(questID) && worldData.CompletedQuests[questID];
    }
}

UI Integration Examples

Save Slot Selection UI

C# 
using System;
using System.Collections.Generic;
using SaveLoadSystem;
using TMPro;
using UnityEngine;
using UnityEngine.UI;

public class SaveSlotSelectionUI : MonoBehaviour
{
    [Header("UI References")]
    [SerializeField] private Transform slotContainer;
    [SerializeField] private GameObject slotPrefab;
    [SerializeField] private Button newSlotButton;
    [SerializeField] private Button deleteModeButton;
    [SerializeField] private TextMeshProUGUI statusText;
    
    [Header("Slot Prefab Components")]
    [SerializeField] private string slotNameInputPath = "NameInput";
    [SerializeField] private string loadButtonPath = "LoadButton";
    [SerializeField] private string deleteButtonPath = "DeleteButton";
    [SerializeField] private string infoTextPath = "InfoText";
    
    private bool deleteMode = false;
    private List<GameObject> currentSlotUIs = new List<GameObject>();
    
    public static event Action<string> OnSlotSelected;
    
    private void Start()
    {
        RefreshSlotList();
        
        newSlotButton.onClick.AddListener(CreateNewSlot);
        deleteModeButton.onClick.AddListener(ToggleDeleteMode);
    }
    
    private void RefreshSlotList()
    {
        // Clear existing UI
        foreach (var slotUI in currentSlotUIs)
        {
            if (slotUI != null) Destroy(slotUI);
        }
        currentSlotUIs.Clear();
        
        // Create UI for each save slot
        var saveSlots = DataManager.GetSaveSlots();
        foreach (var slot in saveSlots)
        {
            CreateSlotUI(slot);
        }
        
        UpdateStatusText();
    }
    
    private void CreateSlotUI(SaveSlot slot)
    {
        GameObject slotUI = Instantiate(slotPrefab, slotContainer);
        currentSlotUIs.Add(slotUI);
        
        // Get components
        var nameInput = slotUI.transform.Find(slotNameInputPath).GetComponent<TMP_InputField>();
        var loadButton = slotUI.transform.Find(loadButtonPath).GetComponent<Button>();
        var deleteButton = slotUI.transform.Find(deleteButtonPath).GetComponent<Button>();
        var infoText = slotUI.transform.Find(infoTextPath).GetComponent<TextMeshProUGUI>();
        
        // Setup slot info
        nameInput.text = slot.Name;
        
        // Show file size and last modified
        long fileSize = slot.GetFileSize();
        DateTime lastModified = slot.GetLastModified();
        string sizeText = fileSize > 0 ? $"{fileSize / 1024f:F1} KB" : "Empty";
        string timeText = lastModified != DateTime.MinValue ? lastModified.ToString("MM/dd/yyyy HH:mm") : "Never";
        infoText.text = $"Size: {sizeText} | Last Modified: {timeText}";
        
        // Setup button events
        loadButton.onClick.AddListener(() => LoadSlot(slot.Name));
        deleteButton.onClick.AddListener(() => DeleteSlot(slot.Name));
        nameInput.onEndEdit.AddListener((newName) => RenameSlot(slot.Name, newName));
        
        // Show/hide delete button based on mode
        deleteButton.gameObject.SetActive(deleteMode);
        
        // Color coding based on file size
        if (fileSize == 0)
        {
            slotUI.GetComponent<Image>().color = new Color(1, 1, 1, 0.5f); // Empty slot
        }
        else if (slot.UseEncryption)
        {
            slotUI.GetComponent<Image>().color = new Color(1, 0.8f, 0.8f, 1); // Encrypted
        }
        else if (slot.UseCompression)
        {
            slotUI.GetComponent<Image>().color = new Color(0.8f, 1, 0.8f, 1); // Compressed
        }
    }
    
    private void CreateNewSlot()
    {
        string baseName = "Save";
        string newName = baseName;
        int counter = 1;
        
        // Find unique name
        while (DataManager.DoesSlotExist(newName))
        {
            newName = $"{baseName}_{counter}";
            counter++;
        }
        
        DataManager.CreateSaveSlot(newName);
        RefreshSlotList();
        
        statusText.text = $"Created new save slot: {newName}";
        statusText.color = Color.green;
    }
    
    private void LoadSlot(string slotName)
    {
        try
        {
            DataManager.SetActiveSlot(slotName);
            OnSlotSelected?.Invoke(slotName);
            
            statusText.text = $"Loaded save slot: {slotName}";
            statusText.color = Color.green;
            
            // Close the selection UI
            gameObject.SetActive(false);
        }
        catch (Exception ex)
        {
            statusText.text = $"Failed to load slot: {ex.Message}";
            statusText.color = Color.red;
        }
    }
    
    private void DeleteSlot(string slotName)
    {
        // Confirmation dialog would be good here
        if (DataManager.GetSaveSlots().Count <= 1)
        {
            statusText.text = "Cannot delete the last save slot";
            statusText.color = Color.red;
            return;
        }
        
        DataManager.DeleteSaveSlot(slotName);
        RefreshSlotList();
        
        statusText.text = $"Deleted save slot: {slotName}";
        statusText.color = Color.yellow;
    }
    
    private void RenameSlot(string oldName, string newName)
    {
        if (string.IsNullOrWhiteSpace(newName) || oldName == newName)
            return;
        
        if (DataManager.DoesSlotExist(newName))
        {
            statusText.text = $"Name '{newName}' already exists";
            statusText.color = Color.red;
            RefreshSlotList(); // Reset UI
            return;
        }
        
        DataManager.RenameSaveSlot(oldName, newName);
        RefreshSlotList();
        
        statusText.text = $"Renamed '{oldName}' to '{newName}'";
        statusText.color = Color.blue;
    }
    
    private void ToggleDeleteMode()
    {
        deleteMode = !deleteMode;
        deleteModeButton.GetComponentInChildren<TextMeshProUGUI>().text = deleteMode ? "Exit Delete Mode" : "Delete Mode";
        deleteModeButton.GetComponent<Image>().color = deleteMode ? Color.red : Color.white;
        
        RefreshSlotList();
    }
    
    private void UpdateStatusText()
    {
        int slotCount = DataManager.GetSaveSlots().Count;
        statusText.text = $"{slotCount} save slot{(slotCount != 1 ? "s" : "")} available";
        statusText.color = Color.white;
    }
}

{
    [Header("UI References")]
    [SerializeField] private GameObject saveIndicator;
    [SerializeField] private TextMeshProUGUI saveText;
    [SerializeField] private Image saveIcon;
    [SerializeField] private Slider progressBar;
    
    [Header("Settings")]
    [SerializeField] private float autoSaveInterval = 30f; // seconds
    [SerializeField] private float indicatorDuration = 2f;
    [SerializeField] private Color savingColor = Color.yellow;
    [SerializeField] private Color savedColor = Color.green;
    [SerializeField] private Color errorColor = Color.red;
    
    private float timeSinceLastSave = 0f;
    private bool isSaving = false;
    
    private void Start()
    {
        saveIndicator.SetActive(false);
        StartCoroutine(AutoSaveRoutine());
    }
    
    private void Update()
    {
        timeSinceLastSave += Time.deltaTime;
        
        // Update progress bar
        if (progressBar != null)
        {
            progressBar.value = timeSinceLastSave / autoSaveInterval;
        }
    }
    
    private IEnumerator AutoSaveRoutine()
    {
        while (true)
        {
            yield return new WaitForSeconds(autoSaveInterval);
            
            if (!isSaving)
            {
                yield return StartCoroutine(PerformAutoSave());
            }
        }
    }
    
    private IEnumerator PerformAutoSave()
    {
        isSaving = true;
        
        // Show saving indicator
        ShowSaveIndicator("Auto-saving...", savingColor);
        
        try
        {
            // Simulate async save operation
            yield return new WaitForSeconds(0.1f);
            
            // Perform actual save operations
            SaveGameData();
            
            // Show success
            ShowSaveIndicator("Auto-save complete!", savedColor);
            timeSinceLastSave = 0f;
        }
        catch (System.Exception ex)
        {
            // Show error
            ShowSaveIndicator($"Auto-save failed: {ex.Message}", errorColor);
            CustomLogger.LogError($"Auto-save failed: {ex.Message}");
        }
        
        // Hide indicator after duration
        yield return new WaitForSeconds(indicatorDuration);
        HideSaveIndicator();
        
        isSaving = false;
    }
    
    public void ManualSave()
    {
        if (!isSaving)
        {
            StartCoroutine(PerformManualSave());
        }
    }
    
    private IEnumerator PerformManualSave()
    {
        isSaving = true;
        
        ShowSaveIndicator("Saving...", savingColor);
        
        try
        {
            yield return new WaitForSeconds(0.1f);
            SaveGameData();
            ShowSaveIndicator("Game saved!", savedColor);
            timeSinceLastSave = 0f;
        }
        catch (System.Exception ex)
        {
            ShowSaveIndicator($"Save failed: {ex.Message}", errorColor);
            CustomLogger.LogError($"Manual save failed: {ex.Message}");
        }
        
        yield return new WaitForSeconds(indicatorDuration);
        HideSaveIndicator();
        
        isSaving = false;
    }
    
    private void SaveGameData()
    {
        // Save all critical game data
        var playerController = FindObjectOfType();
        if (playerController != null)
        {
            playerController.SavePlayerData();
        }
        
        var worldManager = FindObjectOfType();
        if (worldManager != null)
        {
            worldManager.SaveWorldState();
        }
        
        var inventoryManager = FindObjectOfType();
        if (inventoryManager != null)
        {
            inventoryManager.SaveInventory();
        }
        
        // Save timestamp
        DataManager.Save("LastSaveTime", System.DateTime.Now.ToBinary());
    }
    
    private void ShowSaveIndicator(string message, Color color)
    {
        saveIndicator.SetActive(true);
        saveText.text = message;
        saveText.color = color;
        saveIcon.color = color;
        
        // Animate in
        LeanTween.cancel(saveIndicator);
        saveIndicator.transform.localScale = Vector3.zero;
        LeanTween.scale(saveIndicator, Vector3.one, 0.3f).setEaseOutBack();
    }
    
    private void HideSaveIndicator()
    {
        LeanTween.scale(saveIndicator, Vector3.zero, 0.2f).setEaseInBack().setOnComplete(() =>
        {
            saveIndicator.SetActive(false);
        });
    }
    
    private void OnApplicationFocus(bool hasFocus)
    {
        if (!hasFocus && !isSaving)
        {
            // Quick save when losing focus
            StartCoroutine(PerformAutoSave());
        }
    }
    
    private void OnApplicationPause(bool pauseStatus)
    {
        if (pauseStatus && !isSaving)
        {
            // Save when pausing (mobile)
            StartCoroutine(PerformAutoSave());
        }
    }
}

public class EncryptionKeyManager

    public static byte[] GetOrCreateKey()
    {
        string keyString = PlayerPrefs.GetString(KEY_PREF, "");

    }

Migration & Versioning

Handle save file compatibility when updating your game.

C# 
[System.Serializable]
public class SaveFileVersion
{
    public int Version = 1;
    public string GameVersion = "1.0.0";
    public System.DateTime CreatedDate = System.DateTime.Now;
}

public class SaveMigrationManager
{
    private const int CURRENT_SAVE_VERSION = 2;
    
    public static void MigrateIfNeeded()
    {
        var version = DataManager.Load("SaveVersion");
        
        if (version == null)
        {
            // Legacy save file without version
            MigrateFromLegacy();
            version = new SaveFileVersion { Version = CURRENT_SAVE_VERSION };
        }
        else if (version.Version < CURRENT_SAVE_VERSION)
        {
            // Migrate from older version
            MigrateFromVersion(version.Version);
            version.Version = CURRENT_SAVE_VERSION;
        }
        
        DataManager.Save("SaveVersion", version);
    }
    
    private static void MigrateFromLegacy()
    {
        // Example: Convert old player data format
        var oldPlayerData = DataManager.Load("Player");
        if (oldPlayerData != null)
        {
            var newPlayerData = new PlayerData
            {
                Level = oldPlayerData.Level,
                Health = oldPlayerData.HP, // Field name changed
                Position = oldPlayerData.Position
            };
            
            DataManager.Save("PlayerData", newPlayerData);
            // Don't delete old data immediately - keep for rollback
        }
    }
    
    private static void MigrateFromVersion(int fromVersion)
    {
        switch (fromVersion)
        {
            case 1:
                MigrateFrom1To2();
                break;
            // Add more cases as versions increase
        }
    }
    
    private static void MigrateFrom1To2()
    {
        // Example migration logic
        var inventory = DataManager.Load>("Inventory");
        if (inventory != null)
        {
            var newInventory = new List();
            foreach (var itemName in inventory)
            {
                newInventory.Add(new Item { Name = itemName, Quantity = 1 });
            }
            DataManager.Save("PlayerInventory", newInventory);
        }
    }
}

// Use in game initialization
private void Start()
{
    SaveMigrationManager.MigrateIfNeeded();
    LoadGameData();
}
